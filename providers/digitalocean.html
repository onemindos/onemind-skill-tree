How to Connect

◆ doctl CLI & Auth

# Install doctl brew install doctl # Authenticate with API token doctl auth init # → paste your Personal Access Token # List GPU Droplet sizes doctl compute size list | grep gpu

◆ GenAI Platform API

export DO_API_TOKEN="dop_v1_..." # OpenAI-compatible call curl https://api.digitalocean.com/v2/gen-ai/v1/chat/completions \ -H "Authorization: Bearer $DO_API_TOKEN" \ -d '{"model":"llama3.3-70b","messages":[...]}'

◆ Python OpenAI SDK

pip install openai from openai import OpenAI client = OpenAI( api_key="dop_v1_...", base_url="https://api.digitalocean.com/v2/gen-ai/v1" ) resp = client.chat.completions.create( model="llama3.3-70b", messages=[{"role": "user", "content": "Hello!"}] )

◆ GPU Droplet (Self-host)

# Create H100 Droplet doctl compute droplet create gpu-node \ --size gpu-h100x1-80gb \ --image ubuntu-22-04-x64 \ --region nyc3 # Then deploy Ollama on it curl -fsSL https://ollama.com/install.sh | sh ollama pull llama3.3:70b
