Overview

◆ Vertex AI Platform

  • Unified ML platform — training, tuning, deployment
  • Model Garden: access 100+ models from Google & third-parties
  • Pipelines, Feature Store, Vector Search built-in
  • Enterprise SLAs, VPC-SC, CMEK encryption

◆ Gemini API (AI Studio)

  • Direct API access to Gemini family — no GCP project needed
  • Google AI Studio: free-tier prototyping & prompt engineering
  • Zero-to-production in minutes with a single API key
  • Same models as Vertex but simpler auth, fewer controls

◆ Key Differentiators

  • Native multimodal: text, image, video, audio in one model
  • 1M+ token context window (Gemini 1.5 Pro / 2.5 Pro)
  • Grounding with live Google Search — reduces hallucination
  • Tight integration: BigQuery, Cloud Storage, Workspace
multimodal long-context search-grounded enterprise

Available Models

◆ Gemini 2.5 Series

  • gemini-2.5-pro — flagship, deep reasoning, 1M ctx

How to Connect

◆ gcloud CLI & Auth

# Install and authenticate gcloud auth application-default login gcloud config set project YOUR_PROJECT_ID # Set region gcloud config set ai/region us-central1

◆ API Key (AI Studio)

# Get key at aistudio.google.com export GOOGLE_API_KEY="AIza..." # REST call curl -X POST \ "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=$GOOGLE_API_KEY"

◆ Vertex AI SDK (Python)

pip install google-cloud-aiplatform import vertexai from vertexai.generative_models import GenerativeModel vertexai.init(project="my-project", location="us-central1") model = GenerativeModel("gemini-2.5-pro") resp = model.generate_content("Hello!")

◆ Service Account (Prod)

# Create key file gcloud iam service-accounts keys create key.json \ --iam-account=sa@project.iam.gserviceaccount.com export GOOGLE_APPLICATION_CREDENTIALS="./key.json" # Roles needed: # roles/aiplatform.user # roles/ml.developer
