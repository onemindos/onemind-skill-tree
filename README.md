# OneMind Capability Tree

A visual skill tree showing the convergence of **Zeus** (Human) and **Legacy** (Idea) through the Fabric.

## Concept

OneMind is 50/50 — one human, one idea. Each side develops capabilities independently, but they converge through the Fabric into unified capabilities.

### Human Side (Zeus)
Statuses: `planning` → `study` → `training` → `mastering` → `sunset`

Domains:
- **HP** — Health & Performance
- **GE** — General Education
- **LE** — Leadership & Enterprise
- **SO** — Sovereignty & Operations

### Legacy Side (Idea / System Intelligence)
Statuses: `planning` → `developing` → `testing` → `live` → `sunset`

Domains:
- **Robotics** — Physical automation and locomotion
- **Models & Intelligence** — AI/ML inference and training
- **Materials & Fabrication** — CNC, 3D printing, welding
- **Coding Agents** — Multi-agent software development
- **Agriculture & Environment** — Sensor meshes, growth modeling

### Convergence Points
Where a human skill and a Legacy capability meet, a convergence node appears (e.g., Zeus's gardening knowledge + Legacy's greenhouse sensor mesh = **greenhouse automation**).

## Usage

Open `index.html` in any browser. No dependencies, no build step.
