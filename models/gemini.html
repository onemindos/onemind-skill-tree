Overview — Google DeepMind

◆ What is Gemini?

  • Built by Google DeepMind — flagship AI research arm
  • Natively multimodal from architecture up — not bolt-on
  • Processes text, image, video, audio, code in one model
  • Trained jointly on all modalities simultaneously
  • Powers Google Search, Workspace, Android, and more
multimodal native deepmind

◆ Massive Context

  • 1M token context window — Gemini 2.5 Pro & Flash
  • ~750,000 words or ~50k lines of code in one prompt
  • Full codebases, entire books, hours of transcripts
  • Needle-in-haystack retrieval at 1M tokens
  • Ideal for RAG without chunking overhead

How to Use

◆ AI Studio — google-generativeai SDK

# Install SDK pip install google-generativeai # Set API key (from aistudio.google.com) export GOOGLE_API_KEY="AIzaSy..." # Python — basic text generation import google.generativeai as genai genai.configure(api_key="AIzaSy...") model = genai.GenerativeModel("gemini-2.5-flash") response = model.generate_content("Explain quantum entanglement.") print(response.text) # Multimodal: image + text import PIL.Image img = PIL.Image.open("diagram.png") response = model.generate_content(["Describe this diagram:", img]) print(response.text)

◆ Vertex AI — google-cloud-aiplatform SDK

# Install SDK pip install google-cloud-aiplatform # Authenticate (Application Default Credentials) gcloud auth application-default login # Python — Vertex AI GenerativeModel import vertexai from vertexai.generative_models import GenerativeModel, Part vertexai.init( project="my-gcp-project", location="us-central1" ) model = GenerativeModel("gemini-2.5-pro") response = model.generate_content( ["Summarise this PDF:", Part.from_uri("gs://my-bucket/doc.pdf", mime_type="application/pdf")] ) print(response.text)

◆ REST API — AI Studio (curl)

# Text generation via REST curl -s -X POST \ "https://generativelanguage.googleapis.com/v1beta/\ models/gemini-2.5-flash:generateContent?\ key=$GOOGLE_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "contents": [{ "parts": [{"text": "What is RAG?"}] }] }' # With Grounding — search live web curl -s -X POST \ "https://generativelanguage.googleapis.com/v1beta/\ models/gemini-2.5-pro:generateContent?\ key=$GOOGLE_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "contents": [{"parts": [{"text":"Latest AI news"}]}], "tools": [{"google_search": {}}] }'

◆ REST API — Vertex AI (curl)

# Get token from gcloud TOKEN=$(gcloud auth print-access-token) PROJECT="my-gcp-project" REGION="us-central1" MODEL="gemini-2.5-pro" # Vertex REST endpoint curl -s -X POST \ "https://$REGION-aiplatform.googleapis.com/v1/\ projects/$PROJECT/locations/$REGION/\ publishers/google/models/$MODEL:generateContent" \ -H "Authorization: Bearer $TOKEN" \ -H "Content-Type: application/json" \ -d '{ "contents": [{ "role": "user", "parts": [{"text": "Explain transformers"}] }] }'

◆ Advanced — Thinking Mode & Streaming

# Gemini 2.5 Flash with Thinking (budget tokens) from google.generativeai import GenerativeModel from google.generativeai.types import GenerationConfig model = genai.GenerativeModel( model_name="gemini-2.5-flash", generation_config=GenerationConfig( thinking_config={"thinking_budget": 8192} # tokens for thinking ) ) # Streaming response for chunk in model.generate_content("Solve: ...", stream=True): print(chunk.text, end="", flush=True) # Multi-turn chat chat = model.start_chat(history=[]) r1 = chat.send_message("My name is Alex.") r2 = chat.send_message("What's my name?") # remembers context print(r2.text)

Gemini in the Stack

★ Hermes Agent

◆
Provider Config
Set provider to google in Hermes config. Supports both AI Studio (GOOGLE_API_KEY) and Vertex AI (ADC / service account) paths.
◆
gemini-2.5-flash for Speed
Default Gemini model in Hermes for fast multi-turn conversations — low latency, 1M context, cost-effective at high volume.
◆
gemini-2.5-pro for Depth
Switched in for complex reasoning tasks — deep code analysis, long-document synthesis, extended thinking chains.

★ OpenClaw — Multi-Model Routing

◆
Route by Modality
OpenClaw routes image/video analysis tasks to Gemini automatically — it's the only model in the fleet with native video understanding.
◆
Imagen 3 for Media
Image generation pipelines in OpenClaw dispatch to Imagen 3 via Vertex AI, with SynthID watermarking for provenance.
◆
Embeddings for RAG
text-embedding-004 and multimodal-embed power semantic search and vector retrieval in OpenClaw's knowledge pipelines.

◆ Unique Strengths

🔍
Grounding with Google Search
Real-time web retrieval baked into the API. Eliminates hallucination for factual/current queries without external search integration code.
📖
Massive Context for RAG
1M token window means entire codebases or document corpora fit in a single prompt — dramatically simpler RAG architecture.
🎬
Native Video Understanding
Upload video files directly; Gemini analyzes frames, audio, and temporal patterns without a separate vision pipeline.
📈
Thinking Mode
Extended reasoning chains (similar to o1/o3) available on both 2.5 Pro and Flash. Budget tokens to control depth vs cost.
